# Chat App (demo)

### How to run?
1) command -> `npm install`
2) command -> `npm run start`
<br>

<i>Backend:</i>
<ul>
  <li>Nodejs</li> 
  <li>Expressjs</li> 
  <li>Socket.io</li> 
  <li>JWT</li>  
</ul>

<i>Frontend:</i>
<ul>
  <li>Html</li>
  <li>Css & Bootstrap</li>  
  <li>Javascript & JQuery</li> 
</ul>
